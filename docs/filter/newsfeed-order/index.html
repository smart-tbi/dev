Newsfeed Order

The order of the newsfeed can be changed in two ways: Facebook's default order and most recent (chronological order).

Implementation Details​

The implementation of this feature is simple:

  • Default Order navigates the user to Facebook's homepage (https://facebook.com)
  • Most Recent navigates the user to Most Recent feed (https://www.facebook.com/?sk=h_chr).
  • window.location.assign() is used for navigation.
